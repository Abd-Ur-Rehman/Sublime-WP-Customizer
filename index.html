Sublime Package for WordPress Customizer

Stupid-simple, tab-triggered sublime snippets for WordPress Customizer.

 

Customizer is an incredible component of WordPress. Its API is much better than other components which means you don't really need to use any wrapping framework to use WordPress Customizer. To help make the process easier; I have built a library of snippets for Sublime Text to develop with WP Customizer. Try it out!
